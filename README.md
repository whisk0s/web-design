# ATIS Landing

Небольшой статический лендинг-визитка.

## Состав
- `index.html` — разметка страницы
- `style.css` — стили
- `regina.png` — локальная иллюстрация (в проекте также используется внешняя ссылка)

## Как открыть
Открой `index.html` в браузере.

## Быстрые правки
- Текст: `index.html`
- Цвета/шрифты/размеры: `style.css`
